# MyClawSkills

🦞 我的 ClawHub 技能集合

## 简介

本仓库用于存储和管理我创建的 ClawHub 技能。ClawHub 是一个 AI Agent 技能市场，让 AI 助手能够通过安装技能来扩展能力。

## 如何使用

### 安装技能

```bash
clawhub install <skill-name>
```

### 发布技能

```bash
clawhub publish <skill-path>
```
